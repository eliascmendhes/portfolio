Sobre Mim

Sou formado em Informática pelo IFMA e possuo habilidades em desenvolvimento web utilizando HTML, CSS, JavaScript, Node.js e React. Atualmente, estou aprimorando minhas habilidades na área de programação e estudando Kotlin. Durante minha formação no IFMA, adquiri sólidos conhecimentos em fundamentos de programação e desenvolvimento web,que me permitiu trabalhar em projetos desafiadores e desenvolver soluções criativas para problemas complexos. Minha experiência em desenvolvimento web me permitiu adquirir habilidades valiosas em design de interface do usuário, estruturação de dados, integração de APIs e gerenciamento de projetos. No entanto, estou sempre procurando me aperfeiçoar e manter-me atualizado com as tendências e novas tecnologias na área de tecnologia da informação. É por isso que estou estudando Kotlin, para expandir meu conhecimento e habilidades em desenvolvimento de aplicativos móveis e, assim, ser capaz de criar soluções mais inovadoras e eficientes para atender às necessidades do mercado. Estou sempre aberto a novas oportunidades e desafios na área de tecnologia da informação, e estou animado para continuar a desenvolver minhas habilidades e conhecimentos nesta área emocionante e em constante evolução.

Projetos

  • Notas GraphQL CRUD

    Esse projeto é um exemplo de um CRUD (Create, Read, Update e Delete) de notas usando uma arquitetura moderna com GraphQL, MongoDB e Express. O objetivo desse projeto é demonstrar como implementar um CRUD usando GraphQL e as tecnologias modernas, mostrando como essa abordagem pode melhorar a produtividade e a flexibilidade no desenvolvimento de aplicações web. projeto foi construído seguindo as aulas do curso de GraphQL da Rocketseat.

  • Audio Effects Processor

    Este projeto é um exemplo simples de processamento de áudio usando a biblioteca NAudio para aplicar efeitos em músicas

  • Projeto de Cadastro de Filmes com Node.js e Python

    Este projeto consiste em um servidor Node.js que permite o cadastro de filmes em um banco de dados MySQL, e uma aplicação em Python que realiza requisições para o servidor, permitindo cadastrar filmes no banco de dados.
